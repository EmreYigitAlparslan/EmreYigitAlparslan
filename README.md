# Hey, I'm Emre 👋

### Founder • Product Builder • Engineer

I build mobile apps, software products, developer tools, and AI-powered workflows.

I’ve been building since graduating with a computer science degree, and I’ve already had a successful exit — my startup **Drive Buddy** was acquired by Italy-based mobility company **Volvero**.

These days I focus on building practical products fast: things people actually use, not just things that look nice in screenshots.

## 🚀 Selected Projects

### 📊 [Verified Revenues](https://verifiedrevenues.app)
A product focused on revenue transparency and structured business data.

- Built with strict separation between private source data and public-facing data
- Designed as a real product, not just a demo
- Public product, private codebase

### 🌡️ [Claude Meter](https://github.com/EmreYigitAlparslan/claude-meter)
A lightweight macOS menu bar app for tracking Claude Code subscription usage.

- Shows session and weekly usage at a glance
- Built for developers working inside agentic coding workflows
- Open source

### 💌 [Invitation Maker](https://invitationmaker.store/)
A simple product for creating invitations online.

- Consumer-facing product
- Built to be lightweight, accessible, and easy to use
- Live product

### 💧 WaterBot
A Telegram bot for water tracking, reminders, and lightweight habit logging.

- Built as a practical utility for daily use
- Focused on low-friction reminders and tracking
- Private codebase
